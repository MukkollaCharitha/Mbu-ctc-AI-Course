MACHINE LEARNING:-
Machine Learning (ML) is a subset of Artificial Intelligence that allows computers to learn from data and improve their performance without being explicitly programmed.ML algorithms learn patterns from data and make predictions or decisions based on those patterns.

Types of Machine Learning:
Supervised Learning – Uses labeled data ( classification, regression).
Unsupervised Learning – Uses unlabeled data ( clustering).
Reinforcement Learning – Learns by reward and punishment.

Common Algorithms:
Linear Regression
Decision Tree
Random Forest
Support Vector Machine
LightGBM

DEEP LEARNING:-
Deep Learning (DL) is a subset of Machine Learning that uses Artificial Neural Networks with multiple layers (deep neural networks) to learn complex patterns in data.Deep Learning mimics the structure of the human brain using artificial neurons. It automatically extracts features from large amounts of data and performs tasks like image recognition and speech processing

Common Algorithms:
Deep Learning uses Neural Networks, especially:
Artificial Neural Networks (ANN)
Convolutional Neural Networks (CNN)
Recurrent Neural Networks (RNN)

Applications:
Face recognition
Voice assistants
Self-driving cars
Medical image analysis
Language translation

ARTIFICIAL INTELLIGENCE:-
Artificial Intelligence (AI) is a branch of computer science that focuses on creating systems or machines that can perform tasks that normally require human intelligence.AI enables machines to think, learn, reason, solve problems, understand language, and make decisions like humans. It aims to simulate human intelligence using algorithms and data.

Key Features:
Problem solving
Learning from experience
Decision making
Natural language understanding
Image and speech recognition

Examples:
Virtual assistants like Siri
Self-driving cars
Chatbots
Recommendation systems (Netflix, Amazon)
